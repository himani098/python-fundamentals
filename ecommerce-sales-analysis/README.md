# E-Commerce Sales Analysis

## 📌 Project Overview
This project analyzes e-commerce sales data using Python. It demonstrates data cleaning, analysis, and visualization using NumPy, Pandas, and Matplotlib.

## 🛠 Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib

## 📂 Project Structure

```
ecommerce-sales-analysis/
│
├── data/
│   └── ecommerce_sales.csv
├── images/
├── main.py
├── requirements.txt
└── README.md
```

## 📊 Features
- Read data from a CSV file
- Clean missing values
- Create new columns
- Calculate total revenue
- Perform filtering and sorting
- Group data using `groupby()`
- Generate:
  - Bar Chart
  - Pie Chart
  - Line Chart
  - Histogram
  - Scatter Plot
  - Multiple Line Chart
  - Subplots
- Save charts as image files

## ▶️ How to Run

1. Clone the repository
2. Install the required libraries

```bash
pip install -r requirements.txt
```

3. Run the project

```bash
python main.py
```

## 📁 Dataset

The dataset is stored in:

```
data/ecommerce_sales.csv
```

## 📷 Output

The generated charts are saved in the `images` folder.
