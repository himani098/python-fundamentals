Google Colab Notebook Structure

Google Colab notebooks are stored in ".ipynb" format, which is based on JSON.

{
  "cells": [],
  "metadata": {},
  "nbformat": 4,
  "nbformat_minor": 5
}

Explanation

- "cells" : Stores code and markdown cells.
- "metadata" : Stores notebook settings.
- "nbformat" : Notebook format version.
- "nbformat_minor" : Minor notebook version.
